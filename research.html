Expressway Rear-end Conflict Pattern Classification and Modeling

Conflict patterns classification：Classify 7262 conflict events into six conflict patterns using an improved K-Means algorithm adn interpret the meanings of each conflict pattern.

Conflict patterns modeling: Establish a multivariate Poisson-lognormal (MVPLN) model considering spatial-temporal correlation (based on Fixed Rank Kriging) to obtain relationship between the independent variables and the frequency of each conflict pattern in spatial-temporal units.

Tool: Python, R, QGIS.
